🛰️ Print Agents

Cross-network USB printers (e.g. Longer LK5 Pro) driven by an agent on a Raspberry Pi or Windows PC. 📖 Setup guide · 🔒 Security · Architecture

🔑 Cura ingest token

Paste this into the Cura plugin's config.json (and the post-processing uploader). It lets Cura send sliced G-code to the queue.

➕ Add a printer agent

Guided setup creates the agent and generates your ready-to-paste config.json and a one-shot install command for the Pi.

⬆️ Agent software updates

Upload a new agent bundle (a .zip of the printqueue_agent/ package — build with python build_bundle.py). Then use Update agent under each agent's Manage panel — it self-updates and restarts.

⚠️ Printer firmware (advanced)

Upload a Marlin .hex for the printer. Flashing is opt-in per agent (set firmware.enabled in its config) and can brick the board — use with care. Requires avrdude on the agent host.
